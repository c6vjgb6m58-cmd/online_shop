# 前端应用文档

## 环境要求

- Node.js 18+
- npm 或 yarn

## 安装步骤

1. 安装依赖
```bash
npm install
```

2. 启动开发服务器

```bash
npm run dev
```

应用将在 http://localhost:5173 启动

## 构建生产版本

```bash
npm run build
```

构建产物在 `dist` 目录

## 项目结构

```
src/
├── api/           # API调用
├── components/    # 公共组件
├── pages/         # 页面组件
│   ├── admin/     # 管理后台页面
│   └── ...        # 其他页面
├── store/         # 状态管理
├── App.tsx        # 主应用组件
└── main.tsx       # 入口文件
```

## 功能模块

### 顾客端
- 首页：展示热门商品
- 商品列表：浏览、搜索、筛选商品
- 商品详情：查看商品详情，加入购物车
- 购物车：管理购物车商品
- 订单：查看订单列表和详情
- 用户认证：注册、登录、注销

### 管理后台
- 商品管理：添加、编辑、删除商品
- 订单管理：查看订单，更新订单状态
- 客户管理：查看客户信息和行为日志
- 销售统计：查看销售数据和可视化图表
